# Netflix-Trend-Analysis

Welcome to my **Netflix Trend Analysis** project. In this project, I performed **Exploratory Data Analysis (EDA)** and **data preprocessing** on a Netflix dataset to understand content trends, distribution of movies and TV shows, ratings, and yearly release patterns.

The goal of this project is to uncover insights about how Netflix content is distributed across categories, genres, and years using data visualization techniques.

---

## Dataset

The dataset used in this project is the **Netflix Dataset**, which contains information about movies and TV shows available on Netflix.

It includes attributes such as:

- Title
- Category (Movie / TV Show)
- Genre / Type
- Rating
- Release Date
- Other metadata related to Netflix content

This dataset allows analysis of how Netflix content has evolved over time.

## Tools & Libraries

- **Google Colab**
- **Python**
- **Pandas** – Data manipulation and preprocessing
- **NumPy** – Numerical operations
- **Matplotlib** – Data visualization
- **Seaborn** – Statistical data visualization

---

## Project Workflow

1. **Data Loading**
   - Loaded the Netflix dataset using Pandas.

2. **Data Exploration**
   - Checked dataset shape, size, columns, and data types.
   - Generated summary statistics.

3. **Data Cleaning**
   - Removed duplicate records.
   - Identified and visualized missing values using heatmaps.

4. **Content Analysis**
   - Analyzed the number of Movies vs TV Shows.
   - Explored different content types and genres.

5. **Filtering Content**
   - Identified **PG-13 rated movies**.
   - Identified **TV-14 rated TV shows**.

6. **Time-based Analysis**
   - Extracted year from release dates.
   - Analyzed the number of Movies and TV Shows released in **2021**.

7. **Trend Visualization**
   - Visualized **yearly movie releases**.
   - Visualized **yearly TV show releases** using scatter plots.

---

## Key Insights

- Netflix contains a mix of **Movies and TV Shows**, with varying distribution across genres.
- Certain ratings such as **PG-13 and TV-14** appear frequently in Netflix content.
- The number of releases varies significantly across different years.
- Trend analysis helps identify periods where Netflix increased content production.
- Visualization makes it easier to observe **growth patterns in streaming content**.
